1. Tailwind Dark Mode con Soporte Dinámico

Para permitir transiciones perfectas y máxima legibilidad en Modo Claro y Modo Oscuro, configuramos Tailwind de forma explícita basada en clase, mapeando fondos, textos y bordes: 

<script>
    tailwind.config = {
        darkMode: 'class'
    }
</script>

<!-- Estructura adaptable -->
<body class="bg-slate-50 dark:bg-[#080c14] text-slate-800 dark:text-slate-100">
<div class="bg-white dark:bg-slate-900/40 border border-slate-200 dark:border-slate-800/80">

2. Encabezado Dinámico y Logotipo convertidor

Se eliminó la etiqueta de texto fijo y se habilita la edición del 100% de los metadatos corporativos corporales de la organización: 

function saveDocHeader() {
    appDocHeader.appTitle = document.getElementById('app-document-title').innerText;
    appDocHeader.companyName = document.getElementById('company-name').innerText;
    saveLocalData();
}

3. Firma Digital con Trazado Dinámico de Color

El color de la línea del dibujo del canvas ahora se adapta en tiempo real según el tema activo para garantizar máxima legibilidad en fondos blancos u oscuros: 

// Ajustar color del trazo según el tema activo
ctx.strokeStyle = currentTheme === 'dark' ? '#ffffff' : '#1e293b';

function toggleTheme() {
    // ... lógicas del tema
    ctx.strokeStyle = currentTheme === 'dark' ? '#ffffff' : '#1e293b';
}

Cada formulario posee ejemplos de alta fidelidad rellenados por defecto en sus campos de texto, permitiendo un flujo de trabajo intuitivo y demostración de datos de producción listos desde el primer clic.

5. Módulo Modal de Cumplimiento Legal

Cumplimiento estricto con las regulaciones de la República de Colombia (Decreto 1072 de 2015, Resoluciones 0312 de 2019, 1401 de 2007, 4272 de 2021). Incluye lógicas interactivas para abrir un modal modal informativo: 

function showLegalRequirements() {
    document.getElementById('legal-modal').classList.remove('hidden');
}
function closeLegalModal() {
    document.getElementById('legal-modal').classList.add('hidden');
}

    const areaMatch = text.match(/\\[Área:\\s*([\s\S]*?)\\]/);
    if (areaMatch) document.getElementById('area-proceso').value = areaMatch[1].trim();

    saveLocalData();
