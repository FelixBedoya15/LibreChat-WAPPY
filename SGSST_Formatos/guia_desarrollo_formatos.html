Introducción

Esta guía es un manual de usuario detallado diseñado para permitir que cualquier persona, sin necesidad de ser un programador experimentado, cree aplicativos interactivos y avanzados para el Sistema de Gestión de Seguridad y Salud en el Trabajo (SG-SST) utilizando el agente inteligente Consultor SG-SST de WAPPY IA.

¿Cuál es el objetivo final? Obtendrás un único archivo .html premium que podrás abrir en tu computadora de manera offline. Este archivo contará con almacenamiento propio, firma digital, un copiloto conversacional integrado y un tablero Kanban.

B) Conexión con el Copiloto IA (Gemini API) Inteligencia Artificial

Crea una interfaz de chat flotante en el lateral derecho. Utiliza la clave API de Gemini del usuario ingresada en Wappy para llamar al modelo y pasarle el contexto en vivo mediante la función `getDynamicContext()`.

Código Inyectado Estándar: 
function getDynamicContext() {
    return {
        header: appDocHeader,
        wizard: currentWizard, // Todos los inputs del formulario
        ritCompiled: document.getElementById('rit-document-viewer')?.innerText || ""
    };
}

async function askGemini(userMessage) {
    const apiKey = localStorage.getItem('wappy_gemini_key');
    if (!apiKey) return alert("Por favor configura tu API Key en la barra superior o en tu perfil.");
    const model = localStorage.getItem('wappy_gemini_model') || 'gemini-3.5-flash';
    
    const context = JSON.stringify(getDynamicContext());
    const prompt = `Contexto del aplicativo: ${context}\nPregunta: ${userMessage}`;
    
    const response = await fetch(`https://generativelanguage.googleapis.com/v1beta/models/${model}:generateContent?key=${apiKey}`, {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({ contents: [{ role: 'user', parts: [{ text: prompt }] }] })
    });
    const data = await response.json();
    return data.candidates[0].content.parts[0].text;
}
🛠️ ¿Cómo indicarle cambios a la IA?:

Crea una cuadrícula interactiva con tres columnas para registrar las medidas preventivas y correctivas del SG-SST (Plan de Acción). Las tareas se arrastran y sueltan entre las columnas utilizando la API de Drag & Drop de Javascript.

function drag(ev, taskId) {
    ev.dataTransfer.setData("text", taskId);

// Eventos drag y drop nativos en las columnas HTML
function allowDrop(ev) {
    ev.preventDefault();
}

function handleDrop(ev, columnName) {
    ev.preventDefault();
    const taskId = ev.dataTransfer.getData("text");
    const task = currentWizard.tasks.find(t => t.id === taskId);

D) Firmas Fijas (Canvas y Prevención de Carreras) Firmas Digitales

Para evitar que la firma se vuelva invisible al cambiar de tema oscuro a claro, el canvas mantiene un fondo blanco fijo y el trazo es oscuro (`#1e293b`). La bandera `isLoadingState` evita que un canvas vacío pise una firma cargada desde base de datos.

Código Inyectado Estándar: 
let isLoadingState = false; // Bandera antirace

function initializeCanvas(canvasId) {
    const cvs = document.getElementById(canvasId);
    const ctx = cvs.getContext('2d');
    ctx.strokeStyle = '#1e293b'; // Trazo oscuro constante
    ctx.lineWidth = 2.5;
    
    cvs.addEventListener('mouseup', () => {
        if (isLoadingState) return; // Evita pisar durante carga
        saveCanvasToIndexedDB(canvasId, cvs.toDataURL());
    });
}

E) Temas y Colores (Modo Claro/Oscuro) Estilos y Temas

Controla el cambio de apariencia visual inyectando o removiendo la clase `dark` en la etiqueta de documento `html`. Los colores corporativos se definen con clases personalizadas de Tailwind CSS.

Código Inyectado Estándar: 
<script>
function toggleTheme() {
    const htmlElement = document.documentElement;
    const bodyElement = document.body;
    
    if (htmlElement.classList.contains('dark')) {
        htmlElement.classList.remove('dark');
        bodyElement.classList.remove('dark');
        bodyElement.classList.add('light');
        localStorage.setItem('theme-preference', 'light');
    } else {
        htmlElement.classList.add('dark');
        bodyElement.classList.remove('light');
        bodyElement.classList.add('dark');
        localStorage.setItem('theme-preference', 'dark');
    }
}
</script>

A) Control de Firmas (Lienzo Fijo en Modo Claro y Oscuro)

Garantiza que el canvas mantenga un fondo blanco fijo y el trazo sea oscuro `#1e293b` para que la firma siga siendo visible si el usuario cambia el tema del documento o decide imprimirlo: 

function initializeSingleCanvas(cvs) {
    const context = cvs.getContext('2d');
    context.strokeStyle = '#1e293b'; // Trazo oscuro constante
    context.lineWidth = 2.5;
    context.lineCap = 'round';
    context.lineJoin = 'round';
    
    cvs.isDrawing = false;
    cvs.hasDrawn = false;
    
    // Soporte para gestos táctiles y mouse
    cvs.addEventListener('mousedown', (e) => {
        cvs.isDrawing = true;
        context.beginPath();
        const rect = cvs.getBoundingClientRect();
        context.moveTo(e.clientX - rect.left, e.clientY - rect.top);
    });
    
    cvs.addEventListener('mousemove', (e) => {
        if (!cvs.isDrawing) return;
        cvs.hasDrawn = true;
        const rect = cvs.getBoundingClientRect();
        context.lineTo(e.clientX - rect.left, e.clientY - rect.top);
        context.stroke();
    });
    
    cvs.addEventListener('mouseup', () => { cvs.isDrawing = false; saveLocalData(); });
    cvs.addEventListener('mouseleave', () => { cvs.isDrawing = false; });
}
B) Inicialización e Ingesta con IndexedDB

Estructura asíncrona para guardar imágenes y firmas en IndexedDB, previniendo la sobrecarga de cuota de LocalStorage: 

const dbName = 'WappySSTDb';
const dbVersion = 1;
const storeName = 'mediaStore';

function openDB() {
    return new Promise((resolve, reject) => {
        const request = indexedDB.open(dbName, dbVersion);
        request.onupgradeneeded = (e) => {
            const db = e.target.result;
            if (!db.objectStoreNames.contains(storeName)) {
                db.createObjectStore(storeName, { keyPath: 'id' });
            }
        };
        request.onsuccess = (e) => resolve(e.target.result);
        request.onerror = (e) => reject(e.target.error);
    });
}

D) Reemplazo e Inyección para Exportar HTML Autónomo

Función que lee el código fuente del documento, reemplaza los estados por defecto con los datos actuales del LocalStorage, y genera una descarga descargable: 

async function exportUpdatedHTML() {
    saveLocalData();
    let baseHTML = "";
    try {
        const response = await fetch(window.location.href);
        if (response.ok) baseHTML = await response.text();
    } catch (e) {}

    if (!baseHTML) {
        baseHTML = "\n" + document.documentElement.innerHTML + "";
    }

    const headerRegex = /let\s+appDocHeader\s*=\s*\{[\s\S]*?\}\s*;/;
    const wizardRegex = /let\s+currentWizard\s*=\s*\{[\s\S]*?\}\s*;/;

    const newHeaderStr = `let appDocHeader = ${JSON.stringify(appDocHeader, null, 4)};`;
    const newWizardStr = `let currentWizard = ${JSON.stringify(currentWizard, null, 4)};`;

    let updatedHTML = baseHTML;
    if (headerRegex.test(updatedHTML)) updatedHTML = updatedHTML.replace(headerRegex, newHeaderStr);
    if (wizardRegex.test(updatedHTML)) updatedHTML = updatedHTML.replace(wizardRegex, newWizardStr);

    const blob = new Blob([updatedHTML], { type: 'text/html' });
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = appDocHeader.companyName.replace(/[^a-zA-Z0-9]/g, "_") + "_SGSST.html";
    a.click();
}
