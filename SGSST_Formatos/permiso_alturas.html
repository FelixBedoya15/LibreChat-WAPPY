Información Importante de Uso

¿Cómo usar esta herramienta?

Este formato premium de Permiso de Trabajo Seguro en Alturas te permite registrar y autorizar legalmente toda actividad que se realice a 1.50 metros o más sobre el nivel del piso. Utiliza las pestañas superiores para navegar entre los módulos: 1. Identificación y Preoperacional, 2. ATS (Análisis de Trabajo Seguro), 3. Matriz de Riesgos y Peligros, 4. Plan de Acción (Tablero Kanban) y 5. Evidencias y Firmas.

¿Cómo obtener la API Key de Gemini?

Para interactuar con el Copiloto IA de WAPPY, necesitas una API Key de Google Gemini. Puedes conseguirla de forma gratuita siguiendo estos pasos:

  1. Ingresa a Google AI Studio con tu cuenta de Google.
  2. Haz clic en el botón "Get API Key" (Obtener clave de API).
  3. Crea una nueva clave de API para un proyecto nuevo o existente.
  4. Copia la clave y pégala en el campo de configuración del chat lateral (icono de engranaje ⚙️) de este formato, luego presiona "Guardar".

¿Por qué las imágenes, firmas, audios y videos no persisten al cambiar de computador?

El navegador tiene políticas de seguridad estrictas (Sandbox) al abrir archivos HTML directamente desde el disco (protocolo file://).

  • Límite de LocalStorage (5MB): El almacenamiento local estándar es muy limitado. Guardar audios, fotos y videos de alta resolución en él haría que el navegador colapse por exceso de cuota.
  • IndexedDB local: Para evitar colapsos, la herramienta guarda temporalmente las fotos, audios y videos en una base de datos local del navegador (IndexedDB). Esta base de datos es exclusiva de ese equipo y ese navegador.
  • Solución recomendada: Usa el botón "Exportar Permiso de Alturas Firmado" en la pestaña de Firmas para generar un archivo HTML con toda la información y archivos incrustados en Base64. Este archivo exportado es portable y completo.

Marco Legal — Resolución 4272 de 2021

Este formato cumple con los requisitos de la Resolución 4272 de 2021 del Ministerio del Trabajo de Colombia, la cual regula el trabajo en alturas. Es obligatorio que el permiso sea diligenciado, revisado y firmado por un Coordinador de Trabajo en Alturas certificado por el SENA antes de iniciar cualquier actividad a 1.50 m o más. El incumplimiento puede generar sanciones administrativas y penales.

CUMPLIMIENTO LEGAL COLOMBIANO

Debe cumplir strictly con la Resolución 4272 de 2021 de Colombia. Obliga a diligenciar y firmar el permiso de trabajo en alturas de forma presencial por parte del Coordinador de Trabajo en Alturas, garantizando la verificación física del 100% de la lista de chequeo de seguridad.
